# Mesh Dispersal - Interactive 3D Scroll Animation

An interactive WebGL experience featuring scroll-triggered 3D mesh dispersion effects built with Three.js, GSAP, and custom GLSL shaders. Watch as geometric shapes disintegrate into particles as you scroll, with matcap shading and noise-driven animation patterns.
